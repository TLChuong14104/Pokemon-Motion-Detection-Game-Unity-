# Pokémon Motion Detection Game (Unity)

A Pokémon-style game built with Unity, using real-time hand motion detection via webcam to control character movement and actions.

## Features

- Control the character using hand gestures
- No keyboard or controller required
- Python + OpenCV for motion detection
- Unity for gameplay and visuals
